- 🔐 **Bypass age-restricted videos** using cookie-based login

## 🧠 How to Use

1. 🔗 Paste a valid **YouTube**, **Instagram**, or **X (Twitter)** video URL.
2. 🎯 Click **Fetch Video**.
3. 🎚️ Choose your desired quality (e.g., 720p or Audio-only).
4. 💾 Click **Download** and enjoy!
